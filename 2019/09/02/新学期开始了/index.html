新学期开始了

2019-09-02 杂 0 评论 字数统计: 403(字) 阅读时长: 1(分)

新的一学期又开始了，这学期我已经是博三了，这或许将是我学校生活的最后一个学年。时光荏苒，从小学到博士，读书的生活经历了好久，似乎越来越短了。#

博士是学生的尽头么？不，是另一种学习方式的开始。从小学的不知为什么的学习，到高中成绩好的娇傲，到大学被碾压的迷茫、压力突降的颓废，到研究生的随波逐流，到博士前的看不到希望。庆幸的是我已从最低谷的状态慢慢走出来，虽然衣衫褴褛，却心向阳光。当把以前因为学不懂而荒废的功课慢慢捡起来，当看到以前不得要领的知识现在像一个乖宝宝一样静静躺在那里，真的有些说不出的感觉。以前的学习陷入了不想学——学不懂的正反馈环境，无数次地自责与自我怀疑。现在，虽然仍会受到以前的惯性影响，却能大踏步地追赶自己，追赶曾经那个自信而爱学习的自己。我想把自己活成想要的样子。#

最近事情有些不顺利，接二连三的失败，屡败屡战。修修补补，加班学习，把那些落下的知识与时间补回来。明天又是一次面试。昨日的失败让我很难对自己说一句，我已经准备好了。只有不断地提高自己，在学习中慢慢找回自信。加油！！！#
